# Propuesta Académica: BookScan: Gestor de Biblioteca con Escaneo de Portadas

* **Materia:** Integración de Tecnologías
* **Categoría del Proyecto:** Creativos y culturales
* **Complejidad Estimada:** Intermedio

---

### Resumen
Aplicación para organizar tu biblioteca física con solo tomar una fotografía de la portada del libro para catalogarlo de forma automática.

### Módulos a Desarrollar
- Procesamiento de imagen para extraer texto de la portada (OCR).
- Búsqueda automática en la API de Google Books para obtener metadatos (autor, páginas, año).
- Estantería virtual interactiva con categorías personalizadas.
- Registro de libros prestados con alertas de devolución.
- Generador de estadísticas de lectura mensual.

### Stack Tecnológico
Se plantea el uso de las siguientes herramientas: React Native, Node.js, Google Books API, Tesseract OCR

### Utilidad y Justificación Académica
Muestra habilidades en reconocimiento óptico de caracteres e integración con APIs externas en un entorno móvil.
