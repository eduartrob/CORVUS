# Plataforma de Tarjetas de Memoria Inteligentes (FlashMind)

**Categoría:** Educación / Web
**Grupo de Similitud:** Herramientas Educativas
**Nivel de Dificultad:** Fácil - Intermedio

---

## Descripción del Proyecto
Una aplicación web de tarjetas de estudio (flashcards) que utiliza el algoritmo de Repetición Espaciada (SuperMemo-2) para maximizar la retención de memoria a largo plazo.

---

## Características Principales
- Creación de mazos por temas con soporte para texto enriquecido, código e imágenes.
- Algoritmo de repetición espaciada que calcula automáticamente la fecha de próximo repaso.
- Estadísticas diarias de tarjetas repasadas y racha de días de estudio.
- Modo auto-evaluación rápido e interactivo.
- Importación/Exportación masiva de tarjetas en formato CSV.

---

## Stack Tecnológico Sugerido
React, TypeScript, LocalForage, Sass, Material UI
