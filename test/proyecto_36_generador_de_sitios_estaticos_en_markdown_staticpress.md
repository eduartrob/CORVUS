# 🚀 Generador de Sitios Estáticos en Markdown (StaticPress) - Repositorio Escolar

> **Categoría:** DevTools / Web
> **Dificultad:** Fácil - Intermedio
> **Grupo de Similitud:** Herramientas de Programadores

---

## 📖 Descripción General
Una herramienta de línea de comandos en Node.js que convierte de forma veloz carpetas llenas de archivos Markdown (.md) en sitios web HTML listos para ser publicados.

### Estructura Propuesta de Archivos
```
generador_de/
├── backend/            ← Lógica del servidor
├── frontend/           ← Vistas e interfaz de usuario
├── docs/               ← Diagramas y documentación
└── README.md           ← Ficha del proyecto
```

---

## 🎯 Características Principales
- Conversión automática de sintaxis Markdown a etiquetas HTML limpias.
- Soporte integrado para metadatos YAML al inicio del archivo (front-matter).
- Sistema de plantillas personalizables basadas en motores de plantillas como EJS.
- Compilación veloz con optimización integrada de imágenes locales.
- Servidor de desarrollo local integrado con recarga automática del navegador (Live Reload).

---

## 🛠️ Tecnologías Sugeridas
La arquitectura se sustenta en:
`JavaScript, Node.js, EJS, Chokidar (file watcher), Marked (MD parser)`
