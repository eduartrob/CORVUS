# Plataforma de Videoconferencias en el Navegador (TalkSpace)

**Categoría:** Multimedia / Redes
**Grupo de Similitud:** Herramientas de Streaming
**Nivel de Dificultad:** Avanzado

---

## Descripción del Proyecto
Una aplicación web interactiva que facilita las reuniones por video grupales directamente desde el navegador de internet sin complementos externos.

---

## Características Principales
- Transmisión de audio y video HD de igual a igual (Peer-to-Peer) con WebRTC.
- Función interactiva de compartir pantalla para presentaciones o tutoriales.
- Sala de chat integrada para compartir enlaces y notas durante la llamada.
- Pizarra interactiva colaborativa en tiempo real en la que todos pueden dibujar.
- Soporte para grabación de sesiones directamente al disco local del usuario.

---

## Stack Tecnológico Sugerido
JavaScript, React, Node.js (Signaling server), WebRTC, Socket.io, Tailwind CSS
