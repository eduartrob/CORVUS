## 📖 Descripción General
Extensión de navegador que aplica filtros en tiempo real para adaptar los colores de sitios web a distintos tipos de daltonismo.

### Estructura Propuesta de Archivos
```
colorblindwe/
├── backend/            ← Lógica del servidor
├── frontend/           ← Vistas e interfaz de usuario
├── docs/               ← Diagramas y documentación
└── README.md           ← Ficha del proyecto
```

---

## 🎯 Características Principales
- Filtros CSS dinámicos (protanopía, deuteranopía, tritanopía).
- Módulo de análisis de contraste WCAG en tiempo real.
- Ajuste personalizado de saturación y brillo de imágenes.
- Guardado de preferencias del usuario localmente.
- Menú de accesibilidad flotante rápido.
