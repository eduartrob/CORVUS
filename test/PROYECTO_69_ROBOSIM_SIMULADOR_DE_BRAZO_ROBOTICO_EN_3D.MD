# Propuesta Académica: RoboSim: Simulador de Brazo Robótico en 3D

* **Materia:** Integración de Tecnologías
* **Categoría del Proyecto:** Creativos y culturales / Robótica
* **Complejidad Estimada:** Avanzado

---

### Resumen
Entorno virtual interactivo para mover un brazo robótico de 6 ejes y programar secuencias de ensamble mediante bloques visuales.

### Módulos a Desarrollar
- Modelo 3D interactivo con cinemática directa e inversa.
- Lógica de programación visual basada en bloques.
- Simulación física de gravedad y colisiones de objetos.
- Consola de scripting para código embebido.
- Grabación y reproducción de trayectorias físicas.

### Stack Tecnológico
Se plantea el uso de las siguientes herramientas: Three.js, Cannon.js (física), Blockly, HTML5, React

### Utilidad y Justificación Académica
Desarrolla habilidades avanzadas en modelado matemático tridimensional e interfaces dinámicas para estudiantes.
