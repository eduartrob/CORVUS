# Propuesta Académica: Panel de Control de Domótica Residencial (HomeShield)

* **Materia:** Integración de Tecnologías
* **Categoría del Proyecto:** IoT / Domótica
* **Complejidad Estimada:** Avanzado

---

### Resumen
Un panel de control centralizado (Dashboard) para monitorear y controlar dispositivos inteligentes del hogar como luces, termostatos y cámaras de seguridad.

### Módulos a Desarrollar
- Conexión con microcontroladores ESP32 mediante el protocolo MQTT.
- Visualización de temperatura, humedad y consumo eléctrico en tiempo real.
- Interruptores interactivos para encender/apagar luces y electrodomésticos remotos.
- Transmisión de video en vivo de baja latencia desde cámaras de seguridad.
- Módulo de reglas automatizadas (ej. 'Apagar todo si no hay movimiento tras 10 minutos').

### Stack Tecnológico
Se plantea el uso de las siguientes herramientas: Python, Django, Eclipse Mosquitto (MQTT), React, Raspberry Pi, ESP32

### Utilidad y Justificación Académica
Este proyecto es ideal para desarrolladores con un nivel **Avanzado** que buscan enriquecer su portafolio profesional. Al completarlo, habrás dominado conceptos clave relacionados con **IoT / Domótica**, implementando arquitecturas modernas y robustas. ¡Es una excelente adición a tu GitHub!
