# Propuesta Académica: Monitoreo Automatizado de Precios de Vuelos (FlightRadarAlert)

* **Materia:** Integración de Tecnologías
* **Categoría del Proyecto:** Automatización / Viajes
* **Complejidad Estimada:** Intermedio

---

### Resumen
Una aplicación web y bot que monitoriza constantemente los precios de vuelos de aerolíneas específicas para rutas seleccionadas por el usuario, enviando un correo cuando bajan del umbral deseado.

### Módulos a Desarrollar
- Simulación de navegación en navegadores sin cabeza (headless browsers) para evitar bloqueos.
- Gráfica histórica interactiva de la fluctuación de los precios.
- Sistema de alertas configurables mediante SMS (Twilio) o correo electrónico (Nodemailer).
- Panel web multilenguaje para registrar rutas e itinerarios preferidos.
- Manejo avanzado de Proxies rotativos para garantizar la continuidad del servicio.

### Stack Tecnológico
Se plantea el uso de las siguientes herramientas: Node.js, Puppeteer, Express, MongoDB, Nodemailer, Twilio API

### Utilidad y Justificación Académica
Este proyecto es ideal para desarrolladores con un nivel **Intermedio** que buscan enriquecer su portafolio profesional.
