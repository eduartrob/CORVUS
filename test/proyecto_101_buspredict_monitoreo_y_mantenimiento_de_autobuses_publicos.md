# BusPredict: Monitoreo y Mantenimiento de Autobuses Públicos

**Categoría:** Ciencias naturales / Transporte
**Grupo de Similitud:** Sistemas de Transporte Inteligente
**Nivel de Dificultad:** Avanzado

---

## Descripción del Proyecto
Software de análisis de datos para flotas de autobuses que predice fallas mecánicas basándose en lecturas del bus de datos CAN del vehículo.

---

## Características Principales
- Conexión simulada con el puerto OBD-II/CAN del autobús.
- Detección de códigos de error mecánicos (DTC).
- Algoritmo de mantenimiento predictivo (desgaste de frenos, aceite).
- Alertas automáticas de reparación enviadas al taller mecánico central.
- Geolocalización en mapa en tiempo real de unidades con fallas.

---

## Stack Tecnológico Sugerido
Python, InfluxDB, WebSockets, OpenLayers, Raspberry Pi

---

## ¿Por qué elegir este proyecto?
Propuesta avanzada enfocada en Smart Cities, transporte inteligente y optimización de recursos públicos.
