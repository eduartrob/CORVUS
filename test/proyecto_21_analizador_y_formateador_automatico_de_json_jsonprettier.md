# PROTOCOLO DE INVESTIGACIÓN: Analizador y Formateador Automático de JSON (JSONPrettier)

*   **Línea de Investigación:** Ciencias de la Computación / Herramientas
*   **Complejidad de Implementación:** Fácil - Intermedio

---

### 1. Resumen Ejecutivo
Una utilidad multiplataforma que valida archivos JSON, detecta errores de sintaxis y los formatea estéticamente con indentación correcta y colores.

### 2. Requerimientos de Software
- Validador estricto según la especificación RFC 8259.
- Resaltado visual de errores indicando la línea exacta y el caracter faltante.
- Opciones configurables de formato (espacios vs tabs, ordenar llaves alfabéticamente).
- Minificador de JSON integrado para reducir el tamaño de almacenamiento.
- Soporte para procesar archivos de gran tamaño mediante flujos (Streams) sin saturar la memoria.

### 3. Entorno Tecnológico Propuesto
Para llevar a cabo el prototipo, se utilizará: **Python, Tkinter (para interfaz opcional), CLI tools, Sys library**
