# PROTOCOLO DE INVESTIGACIÓN: AgriPrice: Algoritmo de Precios Dinámicos para Cooperativas

*   **Línea de Investigación:** Social / Economía Agraria
*   **Complejidad de Implementación:** Intermedio

---

### 1. Resumen Ejecutivo
Una aplicación web que calcula el precio justo de venta de cosechas agrícolas de pequeños productores basándose en oferta y demanda local.

### 2. Requerimientos de Software
- Carga de inventario por parte de agricultores de la cooperativa.
- Scraping de precios de mercados centrales y centros de abastos.
- Algoritmo que calcula precios mínimos y recomendados de venta.
- Visualizador de tendencias de mercado para planeación de siembra.
- Mapeo de rutas de distribución compartida entre productores.

### 3. Entorno Tecnológico Propuesto
Para llevar a cabo el prototipo, se utilizará: **Node.js, Puppeteer (para scraping), PostgreSQL, React, chart.js**

### 4. Justificación Científica y Técnica
Proyecto con impacto social en el comercio justo, combinando procesamiento de datos, scraping y desarrollo web.
