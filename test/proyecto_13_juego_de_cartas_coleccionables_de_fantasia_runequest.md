PROBLEMÁTICA
La carencia de herramientas automatizadas para Videojuegos / Estrategia genera cuellos de botella y errores frecuentes. Los usuarios dedican demasiado tiempo a procesos repetitivos debido a la falta de centralización y digitalización en sus actividades cotidianas.

OBJETIVO GENERAL
Desarrollar una solución integral para automatizar el flujo de Juego de Cartas Coleccionables de Fantasía (RuneQuest), mejorando significativamente los tiempos de respuesta y la precisión operativa de las tareas asociadas.

OBJETIVOS ESPECÍFICOS
- Diseñar la arquitectura del software contemplando los requerimientos de la categoría Videojuegos / Estrategia.
- Programar e integrar las funcionalidades básicas como: Construcción de mazos personalizados a partir de un inventario de cartas desbloqueables., Mecánicas complejas de combate por turnos (maná, ataque, defensa, habilidades especiales)..
- Implementar controles de seguridad y almacenamiento local robustos.
- Validar la solución ante pruebas de carga y experiencia de usuario.

JUSTIFICACIÓN
Este proyecto se justifica por la necesidad de resolver problemáticas de Videojuegos / Estrategia. Mediante la implementación de Juego de Cartas Coleccionables de Fantasía (RuneQuest), se pretende brindar una alternativa viable y moderna que reduzca los tiempos de trabajo manual y fomente la automatización.

USUARIOS FINALES
- **Administrador del Sistema:** Administra usuarios, configuraciones y emite reportes.
- **Usuario Operativo:** Interactúa con las pantallas operativas principales del sistema.

LISTA DE FUNCIONALIDADES POR TIPO DE USUARIO
**Funcionalidades de Administrador:**
1. Gestión de accesos y monitor de rendimiento.
2. Triage y auditorías.

**Funcionalidades de Usuario Operativo:**
1. Dashboard interactivo.
2. Acceso a las utilidades principales: Construcción de mazos personalizados a partir de un inventario de cartas desbloqueables., Mecánicas complejas de combate por turnos (maná, ataque, defensa, habilidades especiales)..

LISTA DE TECNOLOGÍAS A APLICAR

**Bases de Datos y Otros:** PostgreSQL / SQLite
**Frontend / Interfaz:** Desarrollo responsivo y dinámico.
