# Propuesta Académica: Plataforma de Dibujo Colaborativo en Lienzo (ArtBoard)

* **Materia:** Integración de Tecnologías
* **Categoría del Proyecto:** Arte Digital / Web
* **Complejidad Estimada:** Intermedio

---

### Resumen
Pizarra de dibujo digital interactiva donde múltiples artistas en diferentes partes del mundo pueden pintar simultáneamente sobre el mismo lienzo.

### Módulos a Desarrollar
- Herramientas de dibujo variadas (pincel, lápiz, borrador, formas geométricas, colores personalizados).
- Sincronización instantánea de trazos entre usuarios conectados usando WebSockets.
- Sistema de capas independiente y control de opacidad ajustable por capa.
- Módulo de chat de texto lateral para coordinar el trabajo en grupo.
- Exportación limpia del lienzo completo a formatos PNG o JPG de alta definición.

### Stack Tecnológico
Se plantea el uso de las siguientes herramientas: React, HTML5 Canvas, Socket.io, Express, Tailwind CSS

### Utilidad y Justificación Académica
Este proyecto es ideal para desarrolladores con un nivel **Intermedio** que buscan enriquecer su portafolio profesional. Al completarlo, habrás dominado conceptos clave relacionados con **Arte Digital / Web**, implementando arquitecturas modernas y robustas. ¡Es una excelente adición a tu GitHub!
