# PROTOCOLO DE INVESTIGACIÓN: ToolShare: App P2P para compartir herramientas vecinales

*   **Línea de Investigación:** Social / Economía Circular
*   **Complejidad de Implementación:** Intermedio

---

### 1. Resumen Ejecutivo
Una aplicación móvil comunitaria para compartir y rentar herramientas de construcción de forma descentralizada.

### 2. Requerimientos de Software
- Mapa interactivo de herramientas disponibles en un radio de 5km.
- Sistema de depósito en garantía para protección de equipos.
- Chat integrado entre vecinos con notificaciones push.
- Sistema de reputación y reseñas comunitarias.
- Calendario dinámico de disponibilidad y entrega.

### 3. Entorno Tecnológico Propuesto
Para llevar a cabo el prototipo, se utilizará: **Flutter, Firebase (Auth, Firestore), Google Maps API**

### 4. Justificación Científica y Técnica
Proyecto social y de economía solidaria con gran potencial de uso en comunidades locales.
