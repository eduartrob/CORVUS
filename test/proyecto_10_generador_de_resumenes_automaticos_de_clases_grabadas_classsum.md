# PROTOCOLO DE INVESTIGACIÓN: Generador de Resúmenes Automáticos de Clases Grabadas (ClassSum)

*   **Línea de Investigación:** Inteligencia Artificial / Educación
*   **Complejidad de Implementación:** Avanzado

---

### 1. Resumen Ejecutivo
Herramienta web que procesa grabaciones de audio o video de clases académicas, extrae el texto usando reconocimiento de voz y genera un resumen estructurado con conceptos clave.

### 2. Requerimientos de Software
- Subida de archivos multimedia (mp3, mp4, wav).
- Transcripción automatizada de voz a texto de alta precisión (Whisper API).
- Generación de apuntes estructurados usando modelos LLM.
- Extracción automática de una lista de palabras clave y un cuestionario de estudio.
- Exportación de apuntes a formatos PDF y Markdown.

### 3. Entorno Tecnológico Propuesto
Para llevar a cabo el prototipo, se utilizará: **Python, Django, OpenAI Whisper API, LangChain, React, PostgreSQL**

### 4. Justificación Científica y Técnica
Este proyecto es ideal para desarrolladores con un nivel **Avanzado** que buscan enriquecer su portafolio profesional. Al completarlo, habrás dominado conceptos clave relacionados con **Inteligencia Artificial / Educación**, implementando arquitecturas modernas y robustas. ¡Es una excelente adición a tu GitHub!
