# PROTOCOLO DE INVESTIGACIÓN: CertiChain: Blockchain escolar para títulos académicos

*   **Línea de Investigación:** Educación y ambientes de aprendizaje / Seguridad
*   **Complejidad de Implementación:** Avanzado

---

### 1. Resumen Ejecutivo
Registro estudiantil inmutable para títulos académicos universitarios que evita falsificaciones utilizando contratos inteligentes.

### 2. Requerimientos de Software
- Generación de hashes de certificados escolares.
- Contrato inteligente para emitir y registrar títulos.
- Validador público mediante código QR.
- Portal institucional para firma electrónica de directivos.
- Integración con wallets de estudiantes.

### 3. Entorno Tecnológico Propuesto
Para llevar a cabo el prototipo, se utilizará: **Solidity, Ethereum/Polygon, Ethers.js, React, Node.js**

### 4. Justificación Científica y Técnica
Introduce conceptos clave de criptografía moderna e integridad de datos en el contexto universitario.
