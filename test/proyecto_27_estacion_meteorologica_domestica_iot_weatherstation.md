# Estación Meteorológica Doméstica IoT (WeatherStation)

**Categoría:** IoT / Sensores
**Grupo de Similitud:** Monitoreo y Sensores
**Nivel de Dificultad:** Fácil - Intermedio

---

## Descripción del Proyecto
Dispositivo físico con sensores acoplados que recopila métricas ambientales exteriores e interiores y las envía periódicamente a un servidor web para su análisis.

---

## Características Principales
- Lectura periódica de sensores de temperatura, presión atmosférica y humedad (DHT22 / BMP280).
- Transmisión inalámbrica de datos segura a través de Wi-Fi incorporado.
- Historial gráfico interactivo para ver fluctuaciones a lo largo del año.
- Widget del clima para pantallas LCD físicas conectadas al dispositivo.
- Alertas climáticas extremas automatizadas mediante correos electrónicos.

---

## Stack Tecnológico Sugerido
C++, Arduino IDE, ESP8266 NodeMCU, InfluxDB (Time Series), Grafana
