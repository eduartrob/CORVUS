# PROTOCOLO DE INVESTIGACIÓN: GridSentry: Analizador de Anomalías Eléctricas de Fábricas

*   **Línea de Investigación:** Ciencias naturales / Sistemas Industriales
*   **Complejidad de Implementación:** Avanzado

---

### 1. Resumen Ejecutivo
Software industrial que detecta picos de consumo, cortocircuitos y anomalías en motores de fábricas analizando ondas de corriente.

### 2. Requerimientos de Software
- Procesamiento de muestras de corriente a alta frecuencia.
- Filtro de ruido eléctrico mediante transformada de Fourier.
- Clasificación automática de fallas conocidas (cortos, sobrecargas).
- Notificaciones SMS/Email de emergencia a mantenimiento.
- Visualizador interactivo de armónicos eléctricos.

### 3. Entorno Tecnológico Propuesto
Para llevar a cabo el prototipo, se utilizará: **C++, Python, SciPy, InfluxDB, WebSockets**

### 4. Justificación Científica y Técnica
Una propuesta industrial muy valorada que combina análisis de señales y automatización de mantenimiento.
