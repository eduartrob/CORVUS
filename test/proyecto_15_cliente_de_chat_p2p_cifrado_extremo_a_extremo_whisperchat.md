# Cliente de Chat P2P Cifrado Extremo a Extremo (WhisperChat)

**Categoría:** Ciberseguridad / Comunicación
**Grupo de Similitud:** Herramientas de Red
**Nivel de Dificultad:** Avanzado

---

## Descripción del Proyecto
Un programa de mensajería de escritorio descentralizado que permite chatear directamente con otros usuarios (Peer-to-Peer) utilizando cifrado RSA y AES de grado militar.

---

## Características Principales
- Intercambio seguro de claves criptográficas utilizando el protocolo Diffie-Hellman.
- Mensajería directa cifrada con AES-GCM sin pasar por servidores centrales.
- Generación automática de claves RSA locales para cada sesión.
- Transferencia segura de archivos de igual a igual integrada.
- Interfaz de usuario en terminal minimalista pero elegante.

---

## Stack Tecnológico Sugerido
Rust, tokio (async runtime), ring (cryptography library), libp2p, ratatui
