# PROTOCOLO DE INVESTIGACIÓN: E-commerce de Ropa Vintage (RetroModa)

*   **Línea de Investigación:** Desarrollo Web / E-commerce
*   **Complejidad de Implementación:** Intermedio

---

### 1. Resumen Ejecutivo
Una plataforma web para la compra y venta de ropa vintage y de segunda mano, que incluye un sistema de recomendación inteligente basado en el estilo y las tallas del usuario.

### 2. Requerimientos de Software
- Catálogo interactivo con filtros avanzados por década, estado de conservación, talla y marca.
- Autenticación e inicio de sesión integrados con Firebase Auth.
- Pasarela de pago con Stripe para transacciones seguras.
- Sistema de recomendaciones de prendas personalizadas según el historial de navegación.
- Registro de vendedores independientes con panel de control de ventas e inventario.

### 3. Entorno Tecnológico Propuesto
Para llevar a cabo el prototipo, se utilizará: **React, Node.js, Express, MongoDB, Tailwind CSS, Stripe API**

### 4. Justificación Científica y Técnica
Este proyecto es ideal para desarrolladores con un nivel **Intermedio** que buscan enriquecer su portafolio profesional. Al completarlo, habrás dominado conceptos clave relacionados con **Desarrollo Web / E-commerce**, implementando arquitecturas modernas y robustas. ¡Es una excelente adición a tu GitHub!
