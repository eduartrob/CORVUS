# 🚀 Simulador de Portafolio de Criptomonedas (CryptoSim) - Repositorio Escolar

> **Categoría:** Finanzas / Inversiones
> **Dificultad:** Fácil - Intermedio
> **Grupo de Similitud:** Gestores de Finanzas

---

## 📖 Descripción General
Plataforma interactiva para practicar trading de criptomonedas con dinero virtual utilizando datos de mercado reales e históricos en tiempo real.

### Estructura Propuesta de Archivos
```
simulador_de/
├── backend/            ← Lógica del servidor
├── frontend/           ← Vistas e interfaz de usuario
├── docs/               ← Diagramas y documentación
└── README.md           ← Ficha del proyecto
```

---

## 🎯 Características Principales
- Precios de criptomonedas actualizados al segundo mediante la API de CoinGecko.
- Ejecución de órdenes virtuales de compra, venta y stop-loss.
- Gráficos interactivos de velas japonesas para análisis técnico.
- Clasificación competitiva (leaderboard) entre usuarios según el rendimiento de su cartera.
- Soporte para múltiples monedas fiduciarias de simulación (USD, EUR, MXN).

---

## 🛠️ Tecnologías Sugeridas
La arquitectura se sustenta en:
`React, CoinGecko API, ChartJS, Node.js, MongoDB, WebSockets`
