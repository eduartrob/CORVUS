# Sistema de Recomendación de Películas con Filtrado Colaborativo (MovieSuggest)

**Categoría:** Minería de Datos / Inteligencia Artificial
**Grupo de Similitud:** Sistemas de Recomendación
**Nivel de Dificultad:** Intermedio - Avanzado

---

## Descripción del Proyecto
Una plataforma web que analiza el historial de valoraciones de películas de los usuarios para predecir puntuaciones de películas no vistas y sugerir contenido personalizado utilizando algoritmos de filtrado colaborativo basado en usuarios y en ítems.

---

## Características Principales
- Registro de calificaciones de películas por parte de los usuarios mediante una escala de 1 a 5 estrellas.
- Implementación de algoritmos de vecindario K-Nearest Neighbors (KNN) utilizando la similitud de coseno para asociar usuarios similares.
- Uso de factorización de matrices (Singular Value Decomposition - SVD) para optimizar las predicciones sobre matrices de datos altamente dispersas.
- Panel de control (Dashboard) que muestra listas de recomendaciones personalizadas segmentadas por temáticas y afinidad.
- Módulo estadístico para calcular el error cuadrático medio (RMSE) y el error absoluto medio (MAE) para evaluar el motor.

---

## Stack Tecnológico Sugerido
Python, Pandas, NumPy, scikit-surprise (Surprise Library), FastAPI, React
