# PeerEdit: Editor de Código P2P sin Servidor

**Categoría:** Educación y ambientes de aprendizaje / Redes
**Grupo de Similitud:** Editores P2P
**Nivel de Dificultad:** Avanzado

---

## Descripción del Proyecto
Un editor de código colaborativo en tiempo real que sincroniza los cambios directamente entre los navegadores de los programadores mediante WebRTC.

---

## Características Principales
- Conexión directa P2P (Peer-to-Peer) entre desarrolladores.
- Algoritmo de resolución de conflictos tipo CRDT (Conflict-free Replicated Data Types).
- Chat de voz y texto integrado directamente en el navegador.
- Consola de ejecución de JS remota compartida.
- Soporte para múltiples archivos de código.

---

## Stack Tecnológico Sugerido
JavaScript, WebRTC (Simple-Peer), Yjs (CRDT), Monaco Editor

---

## ¿Por qué elegir este proyecto?
Demuestra dominio en protocolos de red descentralizados y estructuras de datos complejas para sincronización en tiempo real.
