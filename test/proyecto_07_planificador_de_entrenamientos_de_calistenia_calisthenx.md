# PROTOCOLO DE INVESTIGACIÓN: Planificador de Entrenamientos de Calistenia (CalisthenX)

*   **Línea de Investigación:** Salud y Bienestar / Mobile
*   **Complejidad de Implementación:** Fácil - Intermedio

---

### 1. Resumen Ejecutivo
App para móvil que proporciona planes de entrenamiento basados puramente en calistenia (peso corporal), con videos explicativos y rutinas personalizadas.

### 2. Requerimientos de Software
- Rutinas divididas por niveles (principiante, intermedio, avanzado).
- Temporizador interactivo para intervalos, descansos y entrenamientos HIIT.
- Biblioteca con más de 150 ejercicios explicados con animaciones GIF o video cortos.
- Registro de récords personales (máximas dominadas, fondos, flexiones).
- Modo offline para entrenar en parques sin conexión a Internet.

### 3. Entorno Tecnológico Propuesto
Para llevar a cabo el prototipo, se utilizará: **React Native, Expo, SQLite, Lottie Animations, Tailwind CSS**

### 4. Justificación Científica y Técnica
Este proyecto es ideal para desarrolladores con un nivel **Fácil - Intermedio** que buscan enriquecer su portafolio profesional. Al completarlo, habrás dominado conceptos clave relacionados con **Salud y Bienestar / Mobile**, implementando arquitecturas modernas y robustas. ¡Es una excelente adición a tu GitHub!
