# PROTOCOLO DE INVESTIGACIÓN: AgroRecommend: Clasificador de cultivos por sensor de suelo

*   **Línea de Investigación:** Ciencias naturales / Agricultura
*   **Complejidad de Implementación:** Intermedio

---

### 1. Resumen Ejecutivo
Software que analiza pH, humedad y nutrientes del suelo mediante minería de datos para recomendar el cultivo ideal.

### 2. Requerimientos de Software
- Lector de datos de sensores de suelo.
- Modelo de Machine Learning de clasificación (Random Forest).
- Base de datos de requerimientos botánicos de 50 plantas.
- Consejos de fertilización automáticos.
- Exportación de reporte agronómico en PDF.

### 3. Entorno Tecnológico Propuesto
Para llevar a cabo el prototipo, se utilizará: **Python, Scikit-Learn, Pandas, Streamlit, SQLite**

### 4. Justificación Científica y Técnica
Proyecto que une la tecnología moderna con el desarrollo sustentable del campo, ideal para portafolios multidisciplinarios.
