# ScriptEnhance: Escáner y Limpiador de Manuscritos Digitalizados

**Categoría:** Creativos y culturales
**Grupo de Similitud:** Procesamiento de Documentos
**Nivel de Dificultad:** Intermedio

---

## Descripción del Proyecto
Una aplicación web de mejora de imagen diseñada para eliminar manchas, sombras y mejorar el contraste en manuscritos antiguos escaneados.

---

## Características Principales
- Algoritmo de binarización adaptativa para separar texto del fondo con manchas.
- Herramientas manuales de restauración digital de trazo.
- Alineación y enderezamiento automático de páginas torcidas.
- Visor de antes y después interactivo (Slider interactivo).
- Exportación en lotes a formato PDF de alta definición.

---

## Stack Tecnológico Sugerido
React, Python (Flask), OpenCV, NumPy, Tailwind CSS

---

## ¿Por qué elegir este proyecto?
Proyecto técnico enfocado en el procesamiento digital de señales de imagen aplicadas al patrimonio histórico digital.
