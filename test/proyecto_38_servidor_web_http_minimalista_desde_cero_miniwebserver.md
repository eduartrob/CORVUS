## 📖 Descripción General
Un servidor web ligero programado a bajo nivel utilizando sockets nativos para comprender a detalle el protocolo de comunicación HTTP/1.1.

### Estructura Propuesta de Archivos
```
servidor_web/
├── backend/            ← Lógica del servidor
├── frontend/           ← Vistas e interfaz de usuario
├── docs/               ← Diagramas y documentación
└── README.md           ← Ficha del proyecto
```

---

## 🎯 Características Principales
- Escucha activa de conexiones TCP concurrentes mediante hilos de ejecución o I/O multiplexado.
- Lectura y parseo manual de peticiones HTTP (Cabeceras, Métodos GET/POST, Parámetros).
- Envío correcto de respuestas HTTP con tipos MIME adecuados (HTML, CSS, JS, PNG).
- Soporte integrado para manejo de códigos de estado estándar (200 OK, 404 Not Found, 500 Error).
- Sistema básico de enrutamiento estático de archivos del sistema local.

---

## 🛠️ Tecnologías Sugeridas
La arquitectura se sustenta en:
