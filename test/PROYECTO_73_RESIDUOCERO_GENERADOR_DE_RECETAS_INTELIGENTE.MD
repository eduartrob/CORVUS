# Propuesta Académica: Residuocero: Generador de Recetas Inteligente

* **Materia:** Integración de Tecnologías
* **Categoría del Proyecto:** Creativos y culturales / Gastronomía
* **Complejidad Estimada:** Fácil

---

### Resumen
App que genera recetas saludables basándose únicamente en los ingredientes sobrantes en el refrigerador mediante un inventario rápido.

### Módulos a Desarrollar
- Inventario visual y rápido de ingredientes disponibles.
- Motor de búsqueda que prioriza alimentos a punto de caducar.
- Base de datos de recetas caseras con filtros dietéticos.
- Cálculo automático del ahorro económico por comida.
- Compartir recetas improvisadas exitosas con la comunidad.

### Stack Tecnológico
Se plantea el uso de las siguientes herramientas: Vue.js, Express, SQLite, Tailwind CSS

### Utilidad y Justificación Académica
Excelente proyecto introductorio con impacto ecológico de bajo desperdicio de comida.
