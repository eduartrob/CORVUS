# Propuesta Académica: Rastreador de Nutrición y Recetas Inteligentes (NutriSmart)

* **Materia:** Integración de Tecnologías
* **Categoría del Proyecto:** Salud y Bienestar / Mobile
* **Complejidad Estimada:** Intermedio

---

### Resumen
Una aplicación móvil que ayuda a los usuarios a planificar comidas diarias, rastrear calorías y macros, y descubrir recetas según ingredientes disponibles.

### Módulos a Desarrollar
- Escáner de código de barras para productos alimenticios (API de Open Food Facts).
- Diario de alimentos interactivo con calculadora automática de macronutrientes.
- Buscador de recetas que utiliza inteligencia artificial para sugerir qué cocinar.
- Establecimiento de metas personalizadas (pérdida de peso, volumen, mantenimiento).
- Sincronización con dispositivos de salud externos (Apple Health / Google Fit).

### Stack Tecnológico
Se plantea el uso de las siguientes herramientas: Flutter, Dart, Flask (Python backend), SQLite, Open Food Facts API

### Utilidad y Justificación Académica
Este proyecto es ideal para desarrolladores con un nivel **Intermedio** que buscan enriquecer su portafolio profesional. Al completarlo, habrás dominado conceptos clave relacionados con **Salud y Bienestar / Mobile**, implementando arquitecturas modernas y robustas. ¡Es una excelente adición a tu GitHub!
