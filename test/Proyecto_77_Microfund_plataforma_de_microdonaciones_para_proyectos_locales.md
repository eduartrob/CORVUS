# Propuesta Académica: MicroFund: Plataforma de Microdonaciones para Proyectos Locales

* **Materia:** Integración de Tecnologías
* **Categoría del Proyecto:** Social / Finanzas
* **Complejidad Estimada:** Intermedio

---

### Resumen
Un sitio de financiamiento colectivo web para microdonaciones de causas comunitarias y barriales sin intermediarios bancarios.

### Módulos a Desarrollar
- Creación de campañas locales con metas financieras.
- Pasarela de pago modular integrada (MercadoPago/Stripe).
- Muro interactivo de actualizaciones para donantes.
- Visualizador de avances financieros con barras de progreso.
- Validación comunitaria de campañas mediante firmas.

### Stack Tecnológico
Se plantea el uso de las siguientes herramientas: React, Express, MongoDB, Stripe API, CSS Grid

### Utilidad y Justificación Académica
Enseña el manejo de pasarelas de pago, bases de datos no relacionales y lógica de recaudación segura.
