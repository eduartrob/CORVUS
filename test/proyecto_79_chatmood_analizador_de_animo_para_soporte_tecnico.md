# PROTOCOLO DE INVESTIGACIÓN: ChatMood: Analizador de Ánimo para Soporte Técnico

*   **Línea de Investigación:** Ciencias humanas / Utilidades
*   **Complejidad de Implementación:** Intermedio

---

### 1. Resumen Ejecutivo
Filtro automático para clasificar los chats de soporte al cliente según el nivel de frustración o enojo del usuario final.

### 2. Requerimientos de Software
- Análisis léxico y asignación de puntuación de polaridad.
- Detección automática de mayúsculas (gritos) y groserías.
- Priorización automática en la bandeja de entrada para clientes enojados.
- Sugerencias automáticas de respuestas cordiales al agente.
- Estadísticas semanales de satisfacción del departamento.

### 3. Entorno Tecnológico Propuesto
Para llevar a cabo el prototipo, se utilizará: **Python, NLTK, Flask, React, Tailwind CSS**

### 4. Justificación Científica y Técnica
Desarrolla habilidades en procesamiento de lenguaje natural aplicado a la optimización de procesos de negocio.
