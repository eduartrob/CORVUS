# PureWater: Espectrógrafo de bajo costo para Microplásticos

**Categoría:** Ciencias naturales / Ecología
**Grupo de Similitud:** Herramientas de Análisis Químico
**Nivel de Dificultad:** Avanzado

---

## Descripción del Proyecto
Un sistema de análisis de agua que utiliza procesamiento de imágenes microscópicas para detectar la concentración de microplásticos.

---

## Características Principales
- Conexión con microscopio digital USB accesible.
- Procesamiento de imagen por segmentación de color y forma de partículas.
- Cálculo de concentración de partículas por mililitro de agua.
- Base de datos de firmas espectrales ópticas de diferentes plásticos.
- Generador de informes de pureza en formato PDF.

---

## Stack Tecnológico Sugerido
Python, OpenCV, NumPy, Tkinter, ReportLab
