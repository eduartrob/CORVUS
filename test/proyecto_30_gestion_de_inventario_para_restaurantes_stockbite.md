# Gestión de Inventario para Restaurantes (StockBite)

**Categoría:** Gestión de Negocios
**Grupo de Similitud:** Sistemas ERP/Inventario
**Nivel de Dificultad:** Intermedio

---

## Descripción del Proyecto
Herramienta administrativa para restaurantes que permite mantener al día el stock de ingredientes en cocina vinculándolo de forma inteligente con el menú.

---

## Características Principales
- Descuento automático de ingredientes del inventario al vender un platillo del menú.
- Carga manual fácil de facturas de proveedores para reabastecer existencias.
- Alertas tempranas de ingredientes próximos a caducar.
- Cálculo automático del costo de platillo vs precio de venta sugerido.
- Generador de órdenes de compra automáticas en formato PDF para enviar a proveedores.

---

## Stack Tecnológico Sugerido
Laravel, PHP, MySQL, Bootstrap 5, Blade templates
