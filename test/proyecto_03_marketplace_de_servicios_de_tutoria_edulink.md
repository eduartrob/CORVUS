# Marketplace de Servicios de Tutoría (EduLink)

**Categoría:** Desarrollo Web / Plataformas
**Grupo de Similitud:** Tiendas en línea
**Nivel de Dificultad:** Avanzado

---

## Descripción del Proyecto
Un mercado digital interactivo donde estudiantes de todos los niveles pueden buscar, reservar y pagar tutorías personalizadas en tiempo real.

---

## Características Principales
- Calendario interactivo para programar citas integrando la API de Google Calendar.
- Sistema de videollamadas integrado dentro del navegador.
- Soporte para reseñas y calificaciones de tutores.
- Panel de ganancias y cobros automatizados con Stripe Connect.
- Filtro avanzado de tutores por materia, precio y valoraciones.

---

## Stack Tecnológico Sugerido
Next.js, PostgreSQL, Prisma ORM, WebRTC, Google Calendar API, Stripe Connect
