# Plataforma de Voto Electrónico Descentralizado (CryptoVote)

**Categoría:** Blockchain / Seguridad
**Grupo de Similitud:** Blockchain y Cripto
**Nivel de Dificultad:** Avanzado

---

## Descripción del Proyecto
Un sistema de elecciones y votaciones digitales seguro, transparente e inalterable, utilizando contratos inteligentes sobre la blockchain de Ethereum.

---

## Características Principales
- Contrato inteligente robusto escrito en Solidity para registrar votos de forma anónima.
- Verificación de identidad única por votante para evitar la duplicidad de voto.
- Conteo de votos transparente en tiempo real consultando la red Ethereum/Polygon.
- Interfaz de votante web súper intuitiva integrada con billetera MetaMask.
- Cero almacenamiento de datos personales en servidores centralizados.

---

## Stack Tecnológico Sugerido
Solidity, Hardhat, Ethers.js, React, MetaMask integration
