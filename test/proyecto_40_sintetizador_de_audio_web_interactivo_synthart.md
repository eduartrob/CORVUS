# Propuesta Académica: Sintetizador de Audio Web Interactivo (SynthArt)

* **Materia:** Integración de Tecnologías
* **Categoría del Proyecto:** Arte Digital / Audio
* **Complejidad Estimada:** Intermedio

---

### Resumen
Un instrumento musical sintetizador interactivo alojado en una página web que permite a los usuarios diseñar sonidos manipulando osciladores y filtros.

### Módulos a Desarrollar
- Generación nativa de ondas sinusoidales, cuadradas, triangulares y de sierra mediante Web Audio API.
- Teclado musical interactivo en pantalla que responde al teclado físico y a eventos táctiles.
- Efectos de audio ajustables: Eco (Delay), Distorsión, Reverberación y Trémolo.
- Visualizador gráfico del espectro de frecuencia en tiempo real mediante canvas.
- Grabador integrado de audio para exportar creaciones en formato WAV.

### Stack Tecnológico
Se plantea el uso de las siguientes herramientas: HTML5, CSS3, JavaScript, Web Audio API, Sass, Canvas API

### Utilidad y Justificación Académica
Este proyecto es ideal para desarrolladores con un nivel **Intermedio** que buscan enriquecer su portafolio profesional.
