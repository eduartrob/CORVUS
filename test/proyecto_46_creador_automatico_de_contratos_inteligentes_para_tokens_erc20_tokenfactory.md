# PROTOCOLO DE INVESTIGACIÓN: Creador Automático de Contratos Inteligentes para Tokens ERC-20 (TokenFactory)

*   **Línea de Investigación:** Blockchain / FinTech
*   **Complejidad de Implementación:** Intermedio

---

### 1. Resumen Ejecutivo
Plataforma interactiva sin código que permite a emprendedores crear y lanzar sus propios tokens criptográficos en redes compatibles con Ethereum.

### 2. Requerimientos de Software
- Generación dinámica de contratos Solidity con nombre, símbolo, decimales y suministro total.
- Opciones opcionales agregables: Tokenizable (Mintable), Quemable (Burnable) y Pausable.
- Compilación automática del contrato y despliegue directo usando la API de Alchemy.
- Verificación de contrato automatizada en el explorador de bloques de Etherscan.
- Generación automática de un panel de gobernanza básico para el token creado.

### 3. Entorno Tecnológico Propuesto
Para llevar a cabo el prototipo, se utilizará: **React, Solidity, Hardhat, Alchemy API, Etherscan API, Chakra UI**

### 4. Justificación Científica y Técnica
Este proyecto es ideal para desarrolladores con un nivel **Intermedio** que buscan enriquecer su portafolio profesional. Al completarlo, habrás dominado conceptos clave relacionados con **Blockchain / FinTech**, implementando arquitecturas modernas y robustas. ¡Es una excelente adición a tu GitHub!
