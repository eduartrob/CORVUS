# Propuesta Académica: Red Social Temática para Amantes de las Mascotas (PawLife)

* **Materia:** Integración de Tecnologías
* **Categoría del Proyecto:** Redes Sociales / Web
* **Complejidad Estimada:** Intermedio

---

### Resumen
Una red social interactiva dedicada exclusivamente a dueños de mascotas para compartir fotos, consejos, organizar citas de juego en parques y adoptar animales.

### Módulos a Desarrollar
- Creación de perfiles individuales para mascotas con datos de raza, edad y vacunas.
- Feed de publicaciones en tiempo real con fotos, comentarios y me gusta.
- Mapa interactivo para localizar veterinarias de emergencia y parques aptos para perros.
- Módulo de adopción de animales con filtros por ubicación y tipo de mascota.
- Notificaciones de actividad en tiempo real.

### Stack Tecnológico
Se plantea el uso de las siguientes herramientas: React, Node.js, Express, MongoDB, Cloudinary (subida de imágenes), Socket.io

### Utilidad y Justificación Académica
Este proyecto es ideal para desarrolladores con un nivel **Intermedio** que buscan enriquecer su portafolio profesional. Al completarlo, habrás dominado conceptos clave relacionados con **Redes Sociales / Web**, implementando arquitecturas modernas y robustas. ¡Es una excelente adición a tu GitHub!
