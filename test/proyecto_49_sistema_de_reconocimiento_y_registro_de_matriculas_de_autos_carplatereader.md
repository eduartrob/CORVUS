## 📖 Descripción General
Una herramienta de seguridad de bajo costo diseñada para automatizar el control de entradas a estacionamientos privados mediante el escaneo de patentes vehiculares.

### Estructura Propuesta de Archivos
```
sistema_de_r/
├── backend/            ← Lógica del servidor
├── frontend/           ← Vistas e interfaz de usuario
├── docs/               ← Diagramas y documentación
└── README.md           ← Ficha del proyecto
```

---

## 🎯 Características Principales
- Captura de fotogramas estables desde cámaras IP conectadas en red.
- Detección del contorno de vehículos y extracción del área de la patente mediante OpenCV.
- Reconocimiento de caracteres alfanuméricos mediante OCR.
- Búsqueda automática de la patente en una base de datos local de residentes autorizados.
- Apertura remota simulada de portones automáticos registrando historial de accesos.

---

## 🛠️ Tecnologías Sugeridas
La arquitectura se sustenta en:
