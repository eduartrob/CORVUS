# Propuesta Académica: EcoLoop: Plataforma Circular de Residuos Comerciales

* **Materia:** Integración de Tecnologías
* **Categoría del Proyecto:** Social / Economía Circular
* **Complejidad Estimada:** Intermedio

---

### Resumen
Un portal web que conecta comercios locales para que intercambien residuos reciclables (cartón, vidrio, plástico) fomentando el residuo cero.

### Módulos a Desarrollar
- Publicación de residuos industriales/comerciales disponibles por volumen.
- Algoritmo de correspondencia (matchmaking) por cercanía geográfica.
- Módulo de mensajería seguro entre las empresas participantes.
- Cálculo de huella de carbono evitada por cada intercambio exitoso.
- Dashboard para autoridades municipales para ver el mapa de residuos.

### Stack Tecnológico
Se plantea el uso de las siguientes herramientas: Vue.js, Express, MongoDB, Leaflet, Tailwind CSS

### Utilidad y Justificación Académica
Combina desarrollo web moderno con impacto ecológico urbano y soluciones para la economía circular empresarial.
