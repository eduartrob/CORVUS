# Rastreador de Gastos Personales con OCR (SmartPocket)

**Categoría:** Finanzas Personales / Mobile
**Grupo de Similitud:** Gestores de Finanzas
**Nivel de Dificultad:** Intermedio

---

## Descripción del Proyecto
Aplicación móvil diseñada para simplificar el seguimiento financiero diario mediante la lectura inteligente de tickets de compra físicos mediante la cámara.

---

## Características Principales
- Escáner de recibos que digitaliza textos y extrae el costo total, fecha e items.
- Gráficos circulares interactivos y de barra para categorizar los gastos mensuales.
- Establecimiento de presupuestos máximos por categorías (comida, entretenimiento, servicios).
- Sincronización segura de datos en la nube para acceso multidispositivo.
- Exportación directa de balances mensuales a archivos Excel (.xlsx).

---

## Stack Tecnológico Sugerido
React Native, Firebase, Tesseract OCR, Victory Charts, Node.js backend
