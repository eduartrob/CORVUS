# PROTOCOLO DE INVESTIGACIÓN: EcoAir: Red escolar de monitoreo de CO2 y Clima

*   **Línea de Investigación:** Educación y ambientes de aprendizaje / IoT
*   **Complejidad de Implementación:** Intermedio

---

### 1. Resumen Ejecutivo
Red de sensores de calidad de aire en salones para alertar sobre la necesidad de ventilación escolar.

### 2. Requerimientos de Software
- Conexión con sensores de CO2 (MH-Z19) y temperatura.
- Panel web de visualización en la entrada de la escuela.
- Alertas luminosas (LEDs) físicas en los salones.
- Registro de datos históricos para análisis ambiental.
- Exportación de reportes semanales en formato CSV.

### 3. Entorno Tecnológico Propuesto
Para llevar a cabo el prototipo, se utilizará: **ESP8266, MicroPython, MQTT, Node.js, SQLite**

### 4. Justificación Científica y Técnica
Combina hardware escolar accesible, red IoT interna y concientización sobre salud ambiental escolar.
