PROBLEMÁTICA
La carencia de herramientas automatizadas para Ciencias naturales / Agricultura genera cuellos de botella y errores frecuentes. Los usuarios dedican demasiado tiempo a procesos repetitivos debido a la falta de centralización y digitalización en sus actividades cotidianas.

OBJETIVO GENERAL
Desarrollar una solución integral para automatizar el flujo de CropSat: Análisis de Humedad por Imágenes Satelitales, mejorando significativamente los tiempos de respuesta y la precisión operativa de las tareas asociadas.

OBJETIVOS ESPECÍFICOS
- Diseñar la arquitectura del software contemplando los requerimientos de la categoría Ciencias naturales / Agricultura.
- Programar e integrar las funcionalidades básicas como: Conexión con el API de Copernicus Sentinel-2., Cálculo matemático de bandas de luz del índice NDVI..
- Implementar controles de seguridad y almacenamiento local robustos.
- Validar la solución ante pruebas de carga y experiencia de usuario.

JUSTIFICACIÓN
Este proyecto se justifica por la necesidad de resolver problemáticas de Ciencias naturales / Agricultura. Mediante la implementación de CropSat: Análisis de Humedad por Imágenes Satelitales, se pretende brindar una alternativa viable y moderna que reduzca los tiempos de trabajo manual y fomente la automatización.

USUARIOS FINALES
- **Administrador del Sistema:** Administra usuarios, configuraciones y emite reportes.
- **Usuario Operativo:** Interactúa con las pantallas operativas principales del sistema.

LISTA DE FUNCIONALIDADES POR TIPO DE USUARIO
