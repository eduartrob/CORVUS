# PROTOCOLO DE INVESTIGACIÓN: Intérprete de Lenguaje de Programación Propio (Lingo)

*   **Línea de Investigación:** Ciencias de la Computación / Compiladores
*   **Complejidad de Implementación:** Avanzado

---

### 1. Resumen Ejecutivo
Desarrollo completo desde cero de un intérprete para un lenguaje de programación estructurado propio que admita variables, condicionales, bucles y funciones.

### 2. Requerimientos de Software
- Analizador Léxico (Lexer) que desglosa el código fuente en tokens lógicos.
- Analizador Sintáctico (Parser) que construye un Árbol de Sintaxis Abstracta (AST).
- Evaluador recursivo de nodos del AST para ejecutar las instrucciones del código.
- Mensajes de error detallados con números de línea y sugerencias de corrección.
- Interfaz REPL por consola interactiva para probar código en caliente.

### 3. Entorno Tecnológico Propuesto
Para llevar a cabo el prototipo, se utilizará: **Go, Standard Library, Testing packages, CLI tools**

### 4. Justificación Científica y Técnica
Este proyecto es ideal para desarrolladores con un nivel **Avanzado** que buscan enriquecer su portafolio profesional. Al completarlo, habrás dominado conceptos clave relacionados con **Ciencias de la Computación / Compiladores**, implementando arquitecturas modernas y robustas. ¡Es una excelente adición a tu GitHub!
