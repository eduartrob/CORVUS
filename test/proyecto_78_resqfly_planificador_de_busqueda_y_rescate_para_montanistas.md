# PROTOCOLO DE INVESTIGACIÓN: ResQFly: Planificador de Búsqueda y Rescate para Montañistas

*   **Línea de Investigación:** Ciencias naturales / Rescate
*   **Complejidad de Implementación:** Avanzado

---

### 1. Resumen Ejecutivo
Herramienta que define patrones de búsqueda óptimos para drones en terrenos montañosos al buscar excursionistas extraviados.

### 2. Requerimientos de Software
- Generación de trayectorias en espiral y cuadrículas de búsqueda.
- Simulación de elevación del terreno usando datos DEM.
- Algoritmo de optimización de área de cobertura por batería.
- Exportación de plan de vuelo compatible con drones comerciales (QGroundControl).
- Panel de control táctico para rescatistas en campo.

### 3. Entorno Tecnológico Propuesto
Para llevar a cabo el prototipo, se utilizará: **Python, Tkinter, GDAL (datos geoespaciales), GeographicLib**

### 4. Justificación Científica y Técnica
Se enfoca en algoritmos de optimización espacial y manejo de datos geográficos para rescate real.
