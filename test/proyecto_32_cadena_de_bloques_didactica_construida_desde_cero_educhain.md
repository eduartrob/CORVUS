# Cadena de Bloques Didáctica Construida desde Cero (EduChain)

**Categoría:** Blockchain / FinTech
**Grupo de Similitud:** Blockchain y Cripto
**Nivel de Dificultad:** Intermedio

---

## Descripción del Proyecto
Una implementación paso a paso de una blockchain funcional en un entorno local, ideal para fines educativos sobre cómo funcionan los bloques y la criptografía.

---

## Características Principales
- Estructura de datos de Bloque que contiene Hash, Hash anterior, Nonce, Transacciones y Timestamp.
- Algoritmo de Consenso de Prueba de Trabajo (Proof of Work) con dificultad ajustable.
- Cuentas virtuales con firma criptográfica de transacciones mediante pares de claves.
- Consola CLI para simular el minado de bloques y verificar la integridad de la cadena.
- Visualizador web de la cadena para observar bloques, transacciones y balance de direcciones.

---

## Stack Tecnológico Sugerido
TypeScript, Node.js, Crypto library, Express (para API opcional)

---
