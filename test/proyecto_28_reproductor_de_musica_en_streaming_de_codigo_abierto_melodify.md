# Propuesta Académica: Reproductor de Música en Streaming de Código Abierto (Melodify)

* **Materia:** Integración de Tecnologías
* **Categoría del Proyecto:** Multimedia / Web
* **Complejidad Estimada:** Intermedio

---

### Resumen
Una plataforma web auto-alojable de reproducción de música que permite a los usuarios cargar, organizar y reproducir sus bibliotecas de audio personales.

### Módulos a Desarrollar
- Carga de música masiva con extracción automática de metadatos ID3 (carátula, artista, álbum).
- Reproductor de audio continuo sin interrupción (Gapless playback) utilizando Web Audio API.
- Creación y edición interactiva de listas de reproducción dinámicas.
- Búsqueda instantánea mientras se escribe.
- Modo de sincronización de letras de canciones con desplazamiento automático.

### Stack Tecnológico
Se plantea el uso de las siguientes herramientas: React, Express, MongoDB, Howler.js, MusicMetadata parser

### Utilidad y Justificación Académica
Este proyecto es ideal para desarrolladores con un nivel **Intermedio** que buscan enriquecer su portafolio profesional. Al completarlo, habrás dominado conceptos clave relacionados con **Multimedia / Web**, implementando arquitecturas modernas y robustas. ¡Es una excelente adición a tu GitHub!
