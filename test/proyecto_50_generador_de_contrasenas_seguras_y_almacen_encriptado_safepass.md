# Propuesta Académica: Generador de Contraseñas Seguras y Almacén Encriptado (SafePass)

* **Materia:** Integración de Tecnologías
* **Categoría del Proyecto:** Ciberseguridad / Utilidades
* **Complejidad Estimada:** Fácil

---

### Resumen
Un programa local de almacenamiento de credenciales seguro que permite generar contraseñas seguras y guardarlas localmente bajo un esquema de cifrado robusto.

### Módulos a Desarrollar
- Generación aleatoria de contraseñas configurando longitud, caracteres especiales y números.
- Evaluador de la fortaleza de contraseñas en tiempo real basado en entropía de bits.
- Encriptación simétrica AES-256 de base de datos local protegida por una contraseña maestra.
- Función de copiado al portapapeles automático con borrado de seguridad tras 15 segundos.
- Búsqueda rápida por nombre de servicio para un acceso veloz.

### Stack Tecnológico
Se plantea el uso de las siguientes herramientas: Python, cryptography library, SQLite, Tkinter, OS library
