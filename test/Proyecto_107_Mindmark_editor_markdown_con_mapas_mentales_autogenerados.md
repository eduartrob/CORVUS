# MindMark: Editor Markdown con Mapas Mentales Autogenerados

**Categoría:** Educación y ambientes de aprendizaje / Productividad
**Grupo de Similitud:** Editores de Texto
**Nivel de Dificultad:** Intermedio

---

## Descripción del Proyecto
Un editor web de Markdown que analiza la estructura de títulos y viñetas para generar un mapa mental gráfico interactivo a la par.

---

## Características Principales
- Editor markdown de doble panel con previsualización HTML en tiempo real.
- Generación automática del mapa mental (árbol de nodos) a partir del texto.
- Exportación de mapas mentales a imágenes vectoriales SVG o PNG.
- Plantillas de estudio predefinidas (resúmenes, lluvia de ideas).
- Guardado automático e integración de carpetas locales.

---

## Stack Tecnológico Sugerido
React, Mermaid.js o D3.js (para gráficos), Monaco Editor, Node.js

---

## ¿Por qué elegir este proyecto?
Herramienta de productividad visual ideal para estudiantes que ayuda a retener y estructurar ideas complejas.
