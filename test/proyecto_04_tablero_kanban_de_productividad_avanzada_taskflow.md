# Propuesta Académica: Tablero Kanban de Productividad Avanzada (TaskFlow)

* **Materia:** Integración de Tecnologías
* **Categoría del Proyecto:** Productividad / Web
* **Complejidad Estimada:** Intermedio

---

### Resumen
Una aplicación web de tablero de tareas estilo Trello que implementa técnicas avanzadas de organización de proyectos y control de tiempos (Pomodoro).

### Módulos a Desarrollar
- Tableros, listas y tarjetas interactivas mediante drag-and-drop.
- Temporizador Pomodoro integrado por tarjeta para medir el tiempo real dedicado.
- Gráficos analíticos semanales sobre la productividad del usuario.
- Etiquetado inteligente y priorización por colores.
- Exportación de datos del tablero a formatos JSON o CSV.

### Stack Tecnológico
Se plantea el uso de las siguientes herramientas: React, Redux Toolkit, React-beautiful-dnd, Chart.js, LocalFolder Storage

### Utilidad y Justificación Académica
Este proyecto es ideal para desarrolladores con un nivel **Intermedio** que buscan enriquecer su portafolio profesional. Al completarlo, habrás dominado conceptos clave relacionados con **Productividad / Web**, implementando arquitecturas modernas y robustas. ¡Es una excelente adición a tu GitHub!
