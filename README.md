# Landing Front — Corvus

Página de presentación y marketing del proyecto **Corvus — Innovación Académica**.

## Tecnologías
- HTML5 + Tailwind CSS (CDN)
- JavaScript vanilla
- Firebase Hosting (despliegue)

## Estructura
```
landing-front-corvus/
├── index.html              ← Página principal de la landing
├── pages/
│   ├── auth.html           ← Formulario de acceso anticipado
│   ├── early-access.html
│   ├── login.html
│   ├── privacidad.html     ← Política de privacidad (INAI / ARCO)
│   └── terminos.html       ← Términos y condiciones
├── themes/                 ← Tokens de color del sistema de diseño
├── models_mobile_light/    ← Mockups interactivos (tema claro)
├── models_mobile_dark/     ← Mockups interactivos (tema oscuro)
├── examples/               ← Submódulos de referencia de arquitectura AURA
└── script.py               ← Utilidad local de reemplazo de clases CSS
```

## Desarrollo local
Abre `index.html` directamente en el navegador o usa la extensión **Live Server** de VS Code.

## Despliegue
Hospedado en **Firebase Hosting**.
