\subsection{Colección Postman}

Armamos una colección completa en Postman para que el profe o cualquiera pueda darle "Run" y ver que todo jala a la primera.

\begin{itemize}
  \item Archivo en el repo: \texttt{collections/corvus\_clustering.postman\_collection.json}.
  \item Ambiente configurado directo al servidor de producción: \texttt{http://207.180.215.71:3000}.
  \item Casos que probamos sí o sí:
    \begin{enumerate}
      \item \textbf{Pre-validación exitosa (200):} Manda un MD bien hecho y regresa el porcentaje de colisión.
      \item \textbf{Análisis profundo (200):} El endpoint de Ollama con un proyecto real.
      \item \textbf{Historial paginado (200):} Que la ruta \texttt{/inference-history} traiga todo bien paginado.
      \item \textbf{Payload incompleto (422):} Mandar el request sin adjuntar el archivo.
      \item \textbf{Falta sección obligatoria (400):} Mandar un texto que no trae la palabra "Alcance".
      \item \textbf{Draft (200):} Recuperar el último análisis rápido sin volver a procesar nada.
    \end{enumerate}
\end{itemize}
